# Проектная работа 12 (NodeJS с фреймворком Express JS) v0.0.1
Учебный проект Яндекс.Практикум - одностраничный сайт, который подгружает/изменяет профиль пользователя с сервера и загружает с сервера и показывает карточки с фото и названием. К нему добавляем API для загрузки карточек и списка пользователей. В разработке.

Чтобы развернуть проект локально необходимо иметь установленные [Git](https://git-scm.com/) и [Node.js с NPM](https://nodejs.org/en/)

В консоле Git Bush выполните следующие команды:
```
git clone https://github.com/laka-vadim/praktikum-12.git # клонирует данный репозиторий
cd webpack-praktikum # переходим в папку репозитория
npm install # установит все зависимости из package.json
```

Далее запустим сервер на localhost:3000:
```
npm run start # обычный запуск сервера

# ИЛИ

npm run dev # запуск сервера с хотрелоадом - режим разработки
```
